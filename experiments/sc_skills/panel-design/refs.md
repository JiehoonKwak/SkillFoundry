# References

- scGIST: gene panel design for spatial transcriptomics with prioritized gene sets
  URL: https://genomebiology.biomedcentral.com/articles/10.1186/s13059-024-03185-y
  Note: Canonical panel-selection method that balances panel size, cell-type discrimination, and optional priority genes.

- Gene panel selection for targeted spatial transcriptomics
  URL: https://genomebiology.biomedcentral.com/articles/10.1186/s13059-024-03174-1
  Note: `gpsFISH` paper; useful anchor for platform-aware panel selection and cross-platform effects.

- NS-Forest documentation
  URL: https://nsforest.readthedocs.io/en/latest/
  Note: Public documentation for minimum marker combinations, binary specificity scoring, and marker evaluation.

- NS-Forest GitHub repository
  URL: https://github.com/JCVenterInstitute/NSForest
  Note: Reference implementation and release notes for the maintained public package.

- geneBasis: an iterative approach for unsupervised selection of targeted gene panels from scRNA-seq
  URL: https://pmc.ncbi.nlm.nih.gov/articles/PMC8828480/
  Note: Canonical manifold-preserving gene panel method and a useful contrast to quota-based heuristics.

- PanglaoDB: a web server for exploration of mouse and human single-cell RNA sequencing data
  URL: https://pubmed.ncbi.nlm.nih.gov/30951143/
  Note: Public marker compendium for cross-study marker aggregation.

- CellTypist documentation
  URL: https://celltypist.readthedocs.io/en/latest/
  Note: Public reference for marker-informed annotation and cell-type priors.

- Xenium Add-on Panel Design
  URL: https://www.10xgenomics.com/support/in-situ-gene-expression/documentation/steps/panel-design/xenium-add-on-panel-design
  Note: Official platform guidance on selecting genes from tissue-matched single-cell references for targeted spatial panels.
