# References

- Tangram documentation
  URL: https://tangram-sc.readthedocs.io/en/latest/index.html
  Note: Official documentation for Tangram preprocessing, mapping, and gene projection.

- Tangram mapping tutorial with LOOCV
  URL: https://tangram-sc.readthedocs.io/en/latest/tutorial_link.html
  Note: Official tutorial showing training-gene selection, mapping, and leave-one-out validation when the gene panel is small.

- Tangram GitHub repository
  URL: https://github.com/broadinstitute/Tangram
  Note: Canonical implementation surface and tutorial notebooks for `pp_adatas`, `map_cells_to_space`, and `project_genes`.

- Tangram README
  URL: https://raw.githubusercontent.com/broadinstitute/Tangram/master/README.md
  Note: Concise explanation of shared-gene fitting, projected expression, cluster-level mapping, and cosine-similarity loss.

- Tangram paper
  URL: https://www.nature.com/articles/s41592-021-01264-7
  Note: Primary paper describing Tangram for spatial alignment and transcriptome projection.

- AnnData documentation
  URL: https://anndata.readthedocs.io/en/latest/
  Note: Portable `.h5ad` container used by the starter deliverable.

- Squidpy documentation
  URL: https://squidpy.readthedocs.io/en/stable/
  Note: scverse spatial analysis companion package relevant to downstream spatial QC after imputation.
