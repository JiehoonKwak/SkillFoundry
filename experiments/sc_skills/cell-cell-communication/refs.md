# References

- LIANA API documentation
  URL: https://liana-py.readthedocs.io/en/stable/api.html
  Note: Canonical public API reference exposing `geometric_mean` and `rank_aggregate`, which informed the lightweight LIANA-like scoring surrogate.

- LIANA+ paper
  URL: https://www.nature.com/articles/s41556-024-01469-w
  Note: Describes LIANA+ as a unified framework across dissociated and spatial data, including expression filters and permutation-based spatial checks.

- CellPhoneDB results and methods documentation
  URL: https://cellphonedb.readthedocs.io/en/latest/RESULTS-DOCUMENTATION.html
  Note: Defines thresholded interaction means plus empirical shuffling of cluster labels for `statistical_analysis`.

- CellPhoneDB protocol paper
  URL: https://pubmed.ncbi.nlm.nih.gov/32103204/
  Note: Canonical protocol paper for ligand-receptor interaction inference with multi-subunit awareness.

- Squidpy paper
  URL: https://www.nature.com/articles/s41592-021-01358-2
  Note: Documents neighborhood enrichment, co-occurrence, and a CellPhoneDB-style ligand-receptor permutation implementation in spatial context.

- Squidpy graph and neighborhood examples
  URL: https://squidpy.readthedocs.io/en/stable/notebooks/examples/graph/compute_nhood_enrichment.html
  Note: Public example surface for neighborhood-aware spatial validation and adjacency interpretation.
