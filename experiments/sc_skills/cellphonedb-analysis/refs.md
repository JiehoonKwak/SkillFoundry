# References

- CellPhoneDB documentation: analysis methods and output semantics
  URL: https://cellphonedb.readthedocs.io/en/latest/RESULTS-DOCUMENTATION.html
  Note: Canonical description of the statistical label-shuffle method, DEG mode requirements, significant means semantics, and interaction ranking outputs.

- CellPhoneDB GitHub repository
  URL: https://github.com/ventolab/CellphoneDB
  Note: Official package repository with notebooks, example data, and the v5 feature summary used to bound the portable starter against the heavier public stack.

- CellPhoneDB database repository
  URL: https://github.com/ventolab/cellphonedb-data
  Note: Official curated ligand-receptor database repository; the experiment starter replaces this heavy public dependency with a tiny synthetic catalog.

- Efremova M, Vento-Tormo M, Teichmann SA, Vento-Tormo R. CellPhoneDB: inferring cell-cell communication from combined expression of multi-subunit ligand-receptor complexes
  URL: https://pubmed.ncbi.nlm.nih.gov/32103204/
  Note: Canonical CellPhoneDB protocol paper describing curated complexes, mean-expression summaries, and permutation-based specificity testing.
