  URL: https://liana-py.readthedocs.io/en/stable/notebooks/basic_usage.html
  Note: Official LIANA starter usage, including customizable rank aggregation over selected methods.

- LIANA+ paper
  URL: https://www.nature.com/articles/s41556-024-01469-w
  Note: Canonical paper describing the unified single-cell and spatial communication framework and its OmniPath-backed resource model.

- CellPhoneDB analysis methods documentation
  URL: https://cellphonedb.readthedocs.io/en/stable/cellphonedb.src.core.methods.html
  Note: Official API surface for non-statistical and statistical CellPhoneDB analysis modes and their parameterization.

- CellPhoneDB database repository
  URL: https://github.com/ventolab/cellphonedb-data

- OmniPath intercell network documentation
  URL: https://omnipath.readthedocs.io/en/latest/api/omnipath.interactions.import_intercell_network.html
  Note: Official API documentation for importing an intercellular signaling network with transmitter and receiver annotations.

- OmniPath paper
  URL: https://www.embopress.org/doi/full/10.15252/msb.20209923
  Note: Canonical paper for the integrated intra- and intercellular signaling knowledge base.

- Squidpy neighborhood enrichment tutorial
  URL: https://squidpy.readthedocs.io/en/stable/notebooks/examples/graph/compute_nhood_enrichment.html
