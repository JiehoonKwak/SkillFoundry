# References

- Scanpy `tl.dpt` documentation
  URL: https://scanpy.readthedocs.io/en/1.10.x/generated/scanpy.tl.dpt.html
  Note: Official description of diffusion pseudotime, root-cell selection, and the recommendation to use PAGA for branch structure while using DPT for pseudotime.

- PAGA paper
  URL: https://pmc.ncbi.nlm.nih.gov/articles/PMC6425583/
  Note: Canonical graph-abstraction reference for trajectory structure and branch reasoning on single-cell graphs.

- scVelo documentation
  URL: https://scvelo.readthedocs.io/
  Note: Official RNA-velocity documentation covering preprocessing, velocity estimation, and velocity graph ideas.

- scVelo paper
  URL: https://www.nature.com/articles/s41587-020-0591-3
  Note: Primary reference for dynamical RNA velocity modeling and transient-state trajectory inference.

- CellRank tutorials: Kernels
  URL: https://cellrank.readthedocs.io/en/stable/notebooks/tutorials/kernels/
  Note: Official tutorial index for combining velocity and connectivity kernels and for absorbing-state fate analyses.

- CellRank tutorial: CellRank Meets RNA Velocity
  URL: https://cellrank.readthedocs.io/en/stable/notebooks/tutorials/kernels/200_rna_velocity.html
  Note: Official notebook describing transition matrices from RNA velocity plus connectivity and downstream fate probabilities.

- CellRank paper
  URL: https://www.nature.com/articles/s41592-021-01346-6
  Note: Primary reference for directed single-cell fate mapping and absorption probabilities.
