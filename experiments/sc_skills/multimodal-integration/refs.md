# References

- scvi-tools user guide
  URL: https://docs.scvi-tools.org/en/1.3.1/user_guide/index.html
  Note: Canonical documentation for the model families and the tasks they cover, including totalVI and MultiVI.

- totalVI paper
  URL: https://www.nature.com/articles/s41592-020-01050-x
  Note: Primary paper for joint RNA and protein probabilistic modeling in CITE-seq.

- MultiVI paper
  URL: https://www.nature.com/articles/s41592-023-01909-9
  Note: Primary paper for joint RNA and chromatin integration with missing-modality handling.

- Muon input and MuData docs
  URL: https://muon.readthedocs.io/en/latest/io/input.html
  Note: Canonical reference for MuData-style multimodal containers and `.h5mu` workflows.

- Benchmarking algorithms for single-cell multi-omics prediction and integration
  URL: https://www.nature.com/articles/s41592-024-02429-w
  Note: Benchmark context for multimodal integration and prediction method selection.

- Benchmarking single-cell multi-modal data integrations
  URL: https://www.nature.com/articles/s41592-025-02737-9
  Note: More recent benchmark framing across paired, unpaired, and mosaic multimodal settings.
