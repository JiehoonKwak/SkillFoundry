# References

- scvi-tools DestVI tutorial
  URL: https://docs.scvi-tools.org/en/stable/tutorials/notebooks/spatial/DestVI_tutorial.html
  Note: Official tutorial showing the DestVI spatial deconvolution workflow and the gamma-space follow-up analyses that the tiny starter does not attempt to reproduce.

- scvi-tools repository
  URL: https://github.com/scverse/scvi-tools
  Note: Canonical maintained implementation surface for DestVI and related spatial models.

- cell2location estimating signatures notebook
  URL: https://cell2location.readthedocs.io/en/latest/notebooks/cell2location_estimating_signatures.html
  Note: Official walkthrough of reference signature estimation and spatial mapping for the public cell2location workflow.

- cell2location repository
  URL: https://github.com/BayraktarLab/cell2location
  Note: Canonical maintained implementation surface for the Bayesian mapping workflow.

- cell2location paper
  URL: https://www.nature.com/articles/s41587-021-01139-4
  Note: Primary paper for probabilistic cell-type abundance estimation from spatial counts using reference signatures and hierarchical priors.

- DestVI paper
  URL: https://www.nature.com/articles/s41587-022-01272-8
  Note: Primary paper for spatial deconvolution with continuous cell-state variation in the scvi-tools ecosystem.

- Benchmarking spatial and single-cell transcriptomics integration methods for transcript distribution prediction and cell type deconvolution
  URL: https://www.nature.com/articles/s41592-022-01480-9
  Note: Benchmark reference supporting explicit comparison against simple baselines when adapting a tiny starter.

- Advances in mixed cell deconvolution enable quantification of cell types in spatial transcriptomic data
  URL: https://www.nature.com/articles/s41467-022-28020-5
  Note: Review-style reference covering the broader deconvolution problem and motivating lightweight abundance baselines.
