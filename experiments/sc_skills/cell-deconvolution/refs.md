# References

- Tangram getting started
  URL: https://tangram-sc.readthedocs.io/en/latest/getting_started.html
  Note: Official entry point for `pp_adatas`, mapping modes, and the core AnnData workflow surface.

- Tangram GitHub repository
  URL: https://github.com/broadinstitute/Tangram
  Note: Canonical implementation surface for mapping, projection, and constrained-mode utilities.

- Tangram spatial mapping tutorial
  URL: https://tangram-sc.readthedocs.io/en/latest/tutorial_sq_link.html
  Note: Official tutorial showing Tangram mapping over shared genes and Squidpy-linked spatial objects.

- Tangram held-out gene evaluation utilities
  URL: https://tangram-sc.readthedocs.io/en/latest/classes/tangram.plot_utils.plot_auc.html
  Note: Documents the official evaluation path built around `compare_spatial_geneexp` and non-training genes.

- Tangram paper
  URL: https://www.nature.com/articles/s41592-021-01264-7
  Note: Primary paper for Tangram-based mapping, projection, and expression transfer.

- Squidpy tutorials index
  URL: https://squidpy.readthedocs.io/en/stable/notebooks/tutorials/index.html
  Note: Official scverse tutorial index, including the external Tangram deconvolution workflow and segmentation-facing examples.
